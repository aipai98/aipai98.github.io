[Python 安装]

​ 目前，Python有两个版本，一个是2.x版，一个是3.x版，这两个版本是不兼容的。由于3.x版越来越普及，我们的教程将以最新的Python 3.9版本为基础。

  1. 进入Python的官方下载页面
    http://www.python.org/download/

  2. 出现很多版本的，我们选择最新的版本

  3. 下载完成后点击运行，会出现安装界面

    勾选Add Python 3.8 to PATH，这样可以将 Python 命令工具所在目录添加到系统 Path 环境变量中，以后开发程序或者运行 Python 命令会非常方便。

  4. Python 支持两种安装方式，默认安装和自定义安装：

    • 默认安装会勾选所有组件，并安装在 C 盘；
    • 自定义安装可以手动选择要安装的组件，并安装到其它盘符。

    这里我们选择自定义安装，将 Python 安装到常用的目录，避免C盘文件过多。

点击Customize installation进行入下一步，选择要安装的 Python 组件。

没有特殊要求的话，保持默认即可，也就是全部勾选。

点击“Next”继续，选择安装目录。

  1. 设置路径，建议不要设置到C盘，自定义安装可以设置到其它盘，路径越简洁越好，且不要有中文字符

  2. 选择好你常用的安装目录，点击Install，等待几分钟就可以完成安装。

  3. 安装完成以后，打开 cmd窗口，在窗口中输入python命令，如果出现 Python 的版本信息，并看到命令提示符>>>，就说明安装成功了，如果没有识别到python，需要手动配置环境变量

[PyCharm 安装]

  1. Pycharm下载界面：https://www.jetbrains.com/pycharm/download/

  2. 打开安装包，开始安装

  3. 选择安装路径

  4. 功能的选择

  5. 直接下一步，点击Install

  6. 选择稍后重启

  7. 打开Pycharm

  8. 勾选我同意，点击继续

  9. 选择Don't Send

  10. 选择免费试用30天

  11. 创建项目

  12. 点击Create

  13. 右击代码任意部分 ，点击 Run ‘main’，运行第一个程序，出现红色方框结果、成功运行
