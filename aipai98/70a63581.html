IDEA 安装、破解和汉化教程

发表于2023-02-22|更新于2023-02-22
|阅读量:

下载 & 安装

直接到官网下载即可，如下图：

  • Ultimate 即为旗舰版，功能全面，插件丰富，但是收费，按年收费。如果非要比较的话类似于 myEclipse。
  • Community 即为社区版，免费试用，功能相对而言不是很丰富，但是不影响开发使用。如果非要比较的话类似于 eclipse

第二步：卸载老版本 IDEA (未安装则不用管，跳过看下面步骤)

注意，安装新版本 IDEA 之前，如果本机安装过老版本的 IDEA, 需要先彻底卸载，以免两者冲突，导致破解失败。

卸载完成后，点击 Close 按钮关闭弹框：

第三步：安装 IDEA （以 Windows 系统为例）：

卸载老版本 IDEA 完成后，双击刚刚下载好的 IDEA 安装包，弹框会提示选择安装路径，我这里直接选择的默认安装路径，然后点击 Next 按钮：

注意：1. 地址栏中填写的地址不要包含特殊字符

  1. 必须指向一个不存在或内容为空的文件夹！！！

  2. 并不建议安装在 C 盘！

填写完毕后点击 Next 进入下一页面，勾选Create Desktop Shortcut创建 IDEA 桌面快捷启动方式，以方便后面快速打开 IDEA，再点击 Next 按钮 ：

该页面是选择 IDEA 在开始菜单中的文件夹名称的，保持默认即可，点击 Install 按钮，开始安装：

等待安装完成后，不勾选 Run IntellJ IDEA, 点击 Finish 按钮即可完成安装。

破解

下面开始进行本次 IDEA 的破解激活操作。

打开此链接下载破解包：https://pan.baidu.com/s/1C1318411245657

解压下载好的激活包（下载后记得不要删掉，然后安装的路径尽量不要带中文路径、删掉就会失效的）

然后点击打开 scripts 文件

双击启动脚本 unistall-current-user.vbs 直接双击打开。

(此步骤是为了防止之前有过激活信息，确保当前环境变量下没有激活工具的变量信息，可先执行卸载脚本在再进行后面的激活操作，避免激活失败。)

点击后有如下弹出窗，点击确定

出现弹框 done 说明成功。

然后再执行 install-current-user.vbs , 直接双击打开即可。

点击后有如下弹出窗

点击确认后需要等一等 (20 到 40 秒) 然后弹出窗出现 Done

点击确认即可。

先去网站获取最新的激活码，http://jets.idejihuo.com

然后启动 IDEA，这里采用的是 Activation Code 破解

IDEA 首次运行会弹出一个注册框，勾选 Evaluate for free，点击 Evaluate，然后进入主界面

在 Activation Code 框内填入在网站里获取到的激活码即可

激活完成

然后就好了，时间是假的，已经永久激活了，而且是官网最新版本哦，如果你重新更新了，再执行上面的操作即可。

常见问题

在破解 idea 的时候，输入激活码，出现如下提示：

key is invalid

原因：

  1. 执行破解脚本提示 Done, 检查配置环境变量的值：路径中是否包含中文，包含中文是不行的，换成英文路径

  2. 我这边的破解教程，亲测的版本 2022.2、2022.1、2021.3.3、2021.3.2 都是可以的，确认你的版本号，太老的版本或太新的版本都不行

  3. 有的可以重启电脑，然后在输入就成功了

  4. 检查破解补丁的位置是否动了，切记不要动，不然重启 IDE 又找不到补丁位置了，自然就失败了

  5. 最后终结解决办法：在控制面板完整的卸载掉 idea，然后重新安装 idea，最后重启电脑，然后按照下方的破解教程重新来一遍即可。

汉化

进入IDEA主界面，点击 File–>Settings

点击左侧的 Plugins 按钮，然后点击 Marketplace，然后在搜索框搜索 chinese 安装图中所示的插件：

找到下载插件，选择勾选上，然后 ok

ok 之后会弹出一个提示，让你重启，重启之后就可以了
